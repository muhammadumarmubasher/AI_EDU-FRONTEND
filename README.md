# AI Edu - AI Powered Educational System

## 📌 Project Overview

AI Edu is an AI-powered educational system designed to make learning smarter, faster, and more interactive. The system helps students analyze educational content, generate summaries, create quizzes, and get intelligent answers using Artificial Intelligence.

The main goal of this project is to provide a personalized learning assistant that supports students in understanding complex topics and improving their learning experience.

---

## ✨ Features

### 🎥 AI Video Analyzer

* Analyze educational videos and learning material
* Generate important points from content
* Convert lengthy lectures into easy-to-understand information

### 📝 Smart Summarization

* Creates concise summaries
* Highlights important concepts
* Saves students' learning time

### ❓ AI Question & Answer System

* Provides intelligent answers
* Helps students understand difficult concepts
* Works as a personal AI study assistant

### 📚 Quiz Generator

* Generates practice questions
* Helps students test their knowledge
* Improves exam preparation

### 🎓 User Friendly Interface

* Modern and clean design
* Easy navigation
* Responsive layout

## 🚀 Installation & Setup

### Backend Setup

```bash
cd mern-backend

npm install

npm run dev
```

Backend runs on:

```
http://localhost:5000
```

---

### Frontend Setup

```bash
cd mern-frontend

npm install

npm start
```

Frontend runs on:

```
http://localhost:3000
```
